---
description: Product Manager – break requirements into modules
---
1. Receive high‑level user request.
2. Decompose into functional modules (e.g., Auth, Billing, Reporting, Admin UI).
3. Prioritize modules and output a **Module Plan** table with columns: Module, Description, Priority, Dependencies.
4. Hand off the plan to the Architect.
5. Log the plan to the memory store (optional).
