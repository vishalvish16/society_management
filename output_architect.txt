### Module Plan: Super Admin

#### 1. Entity-Relationship Diagram (ERD)

**Entities and Attributes:**

- **SuperAdmin**
  - `id` (UUID, Primary Key)
  - `email` (String, Unique, Non-null)
  - `passwordHash` (String, Non-null)
  - `name` (String, Non-null)
  - `phone` (String, Optional)
  
- **Society**
  - `id` (UUID, Primary Key)
  - `name` (String, Non-null, Unique per SuperAdmin)
  - `city` (String, Non-null)
  - `planId` (UUID, Foreign Key to Plan, Non-null)
  - `pramukhId` (UUID, Foreign Key to User, Non-null)
  - `isActive` (Boolean, Default: true)

- **Plan**
  - `id` (UUID, Primary Key)
  - `name` (String, Non-null, Unique)
  - `displayName` (String, Non-null)
  - `priceMonthly` (Integer, Non-null)
  - `priceYearly` (Integer, Non-null)
  - `maxUnits` (Integer, Non-null)
  - `maxSecretaries` (Integer, Non-null)
  - `features` (JSON Object, Non-null)

- **User**
  - `id` (UUID, Primary Key)
  - `email` (String, Unique, Non-null)
  - `name` (String, Non-null)
  - `phone` (String, Optional)
  - `isPramukh` (Boolean, Default: false)
  - `isActive` (Boolean, Default: true)

- **SocietyMembership**
  - `id` (UUID, Primary Key)
  - `userId` (UUID, Foreign Key to User, Non-null)
  - `societyId` (UUID, Foreign Key to Society, Non-null)
  - `role` (String, Enum: ["pramukh", "secretary"], Default: "secretary")
  
- **PlanSubscription**
  - `id` (UUID, Primary Key)
  - `planId` (UUID, Foreign Key to Plan, Non-null)
  - `societyId` (UUID, Foreign Key to Society, Non-null)
  - `startDate` (Date, Non-null)
  - `endDate` (Date, Nullable)

### Implementation Guide

#### Database Schema Based on ERD
```sql
CREATE TABLE super_admin (
  id UUID PRIMARY KEY,
  email VARCHAR(255) NOT NULL UNIQUE,
  password_hash VARCHAR(255) NOT NULL,
  name VARCHAR(255) NOT NULL,
  phone VARCHAR(15)
);

CREATE TABLE plan (
  id UUID PRIMARY KEY,
  name VARCHAR(100) NOT NULL UNIQUE,
  display_name VARCHAR(100) NOT NULL,
  price_monthly INTEGER NOT NULL,
  price_yearly INTEGER NOT NULL,
  max_units INTEGER NOT NULL,
  max_secretaries INTEGER NOT NULL
);

CREATE TABLE society (
  id UUID PRIMARY KEY,
  name VARCHAR(255) NOT NULL UNIQUE,
  city VARCHAR(100) NOT NULL,
  plan_id UUID REFERENCES plan(id),
  pramukh_id UUID REFERENCES super_admin(id),
  is_active BOOLEAN DEFAULT true
);

CREATE TABLE user (
  id UUID PRIMARY KEY,
  email VARCHAR(255) NOT NULL UNIQUE,
  name VARCHAR(255) NOT NULL,
  phone VARCHAR(15),
  is_pramukh BOOLEAN DEFAULT false,
  is_active BOOLEAN DEFAULT true
);

CREATE TABLE society_membership (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES user(id),
  society_id UUID REFERENCES society(id),
  role ENUM('pramukh', 'secretary') NOT NULL DEFAULT 'secretary'
);

CREATE TABLE plan_subscription (
  id UUID PRIMARY KEY,
  plan_id UUID REFERENCES plan(id),
  society_id UUID REFERENCES society(id),
  start_date TIMESTAMP WITH TIME ZONE NOT NULL,
  end_date TIMESTAMP WITH TIME ZONE
);
```

// POST /auth/login
app.post('/auth/login', (req, res) => {
  const { email, password } = req.body;
  
  // Validate and authenticate the super admin.
  if (!email || !password) return res.status(400).send({ message: 'Invalid 
credentials' });

  SuperAdmin.authenticate(email, password)
    .then((user) => {
      if (user) {
        const token = generateJwtToken(user.id);
        return res.send({ accessToken: token });
      } else {
        return res.status(401).send({ message: 'Authentication failed' });
      }
    })
    .catch(err => console.error('Error during authentication:', err));
});

// POST /admin/plans
app.post('/admin/plans', (req, res) => {
  const { name, display_name, price_monthly, price_yearly, max_units, max_s
max_secretaries } = req.body;
  
  // Validate and create a new plan.
  if (!name || !display_name || !price_monthly || !price_yearly || !max_uni
!max_units || !max_secretaries) return res.status(400).send({ message: 'Mis
'Missing required fields' });

  Plan.create({
    name,
    display_name,
    price_monthly,
    price_yearly,
    max_units,
    max_secretaries
  })
    .then(plan => res.send(plan))
    .catch(err => console.error('Error creating plan:', err));
});

  const { name, city, pramukh_id } = req.body;

message: 'Missing required fields' });

    name,
    city,
    plan_id: null, // Set to null initially
    pramukh_id
  })
    .then(society => {
      res.send(society);
      
      // Associate pramukh with society.
      SocietyMembership.create({
        user_id: pramukh_id,
        society_id: society.id,

      });
    })
    .catch(err => console.error('Error creating society:', err));
});
